# Code_Alpha_EDA
This EDA repository for CodeAlpha's Data Analytics Task 2 parses the Titanic dataset using Pandas, NumPy, Matplotlib, and Seaborn. The code checks shapes, finds missing percentages, uses IQR to isolate outliers, and tests survival across classes and genders. It concludes by auto-saving a 6-plot visual matrix as a high-res PNG file.
